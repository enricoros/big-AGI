Inspect
Standalone developer tools. Static pages with zero app imports, safe to open even when big-AGI is broken.

Low-Level tools

/dev/inspect/storagePer-key sizes across localStorage, IndexedDB, OPFS and quota, with store sub-key breakdowns. /dev/inspect/logsThe client log buffer with level and source filters, plus search. /dev/inspect/doctorOne-glance PASS/FAIL of the things that break boot.

App tools (require the running app)

/dev/llm-capabilitiesFunction-calling capability matrix across configured vendors. /info/debugClient capabilities, product state, and cloud info. Downloadable JSON.
These pages read only your own browser data and run entirely on your device.
