Dev tools
Developer tools for big-AGI. The Inspect tools are static pages with zero app imports, safe to open even when big-AGI is not working properly; the app tools need the running app.

Inspect - static

/dev/inspect/storagePer-key sizes across localStorage, IndexedDB, OPFS and quota, with store sub-key breakdowns. /dev/inspect/logsThe client log buffer with level and source filters, plus search. /dev/inspect/doctorOne-glance PASS/FAIL of the things that break boot.

App tools - require the running app

/dev/llmsFunction-calling capability matrix across configured vendors. /dev/debugClient capabilities, product state, and backend/cloud info. Downloadable JSON.
The Inspect tools read only your own browser data and run entirely on your device. The app tools run in big-AGI - and /dev/llms sends probes to your configured model providers.
